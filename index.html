E-commerce Image Toolkit

Product Visualizer

A collection of Python utilities that automate the boring parts of e-commerce product imagery — removing backgrounds, resizing in bulk, scraping and downloading catalog images, and stamping campaign stickers straight from Photoshop.

How it works

Every tool follows the same simple, folder-based workflow.

01

Install Python

Grab Python 3.9+ and install each tool's dependencies with pip install.

02

Add your files

Drop images into the input folder, or point the scraper at your .xlsx / .csv.

03

Run the script

Double-click the run.bat or run python <script>.py from a terminal.

04

Collect output

Processed images and updated spreadsheets land in the output folder.

Note: these are desktop Python tools — they run on your own machine, not in the browser. This page is the project's live showcase and documentation. Head to the GitHub repository to download and run any tool.
